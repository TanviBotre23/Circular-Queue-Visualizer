Circular Queue Visualizer

A Circular Queue is a linear data structure following FIFO order where the last position connects back to the first using modulo arithmetic.

Front: -1

Rear: -1

Circular Queue Algorithm

1. Enqueue Algorithm

  1. Check overflow condition: if (rear + 1) % size == front
  2. If queue is empty, set front = 0
  3. Move rear using modulo arithmetic
  4. Insert element at queue[rear]
void enqueue(int value) {
    if ((rear + 1) % size == front)
        return; // Overflow

    if (front == -1)
        front = 0;

    rear = (rear + 1) % size;
    queue[rear] = value;
}

2. Dequeue Algorithm

  1. Check underflow condition: if front == -1
  2. Store element at front
  3. If front == rear, reset both to -1
  4. Else move front using modulo arithmetic
int dequeue() {
    if (front == -1)
        return -1; // Underflow

    int value = queue[front];

    if (front == rear)
        front = rear = -1;
    else
        front = (front + 1) % size;

    return value;
}
