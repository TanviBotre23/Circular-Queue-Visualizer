# Circular-Queue-Visualizer
Circular Queue Visualizer is a DSA mini-project that demonstrates the working of a Circular Queue data structure through an interactive web-based visualizer. It includes a C++ backend implementation and a frontend built with HTML, CSS, and JavaScript that animates enqueue and dequeue operations in real time on a circular display.


## 📌 Project Overview
A DSA project that visually demonstrates the Circular Queue data structure.
It includes a working C++ implementation and an interactive web-based
frontend visualizer that shows real-time queue operations.

## ⚙️ Features
- Create a circular queue of custom size (min: 3)
- Enqueue and Dequeue elements with visual feedback
- Circular display with front (green) and rear (orange) pointers
- Overflow and Underflow detection with messages
- Algorithm pseudocode panel alongside the visualizer
- Fully responsive layout

---

## 🖥️ How to Run

### Frontend (Web Visualizer)
1. Open the `frontend/` folder
2. Open `index.html` in any browser
3. No server or installation required

### Backend (C++ Program)
1. Open terminal in `backend/` folder
2. Compile:
   g++ circular_queue.cpp -o circular_queue
3. Run:
   ./circular_queue   (Linux/Mac)
   circular_queue.exe (Windows)
